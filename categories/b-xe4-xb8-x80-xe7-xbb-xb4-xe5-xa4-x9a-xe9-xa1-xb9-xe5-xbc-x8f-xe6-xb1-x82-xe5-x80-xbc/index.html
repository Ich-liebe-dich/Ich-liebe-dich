一维多项式求值

这里提供两种方法求一维多项式的值

一、代入法

顾名思义，就是将未知量的值代入多项式中直接求解。同样的，我们还是用数组表示多项式的各个系数，用数组计算。

double polynomial_1(double A[], int n, double x)
{
    int i;
    double result = A[0];
    for (i = n - 1; i >= 1; i--)
    {
        result += A[i] * pow(x, i);
    }
    return result;
}

二、秦九韶算法

秦九韶算法是中国南宋时期的数学家秦九韶提出的一种多项式简化算法。

对于一个n次多项式：

我们可以这样化简：

它变成了若干次的加法与乘法运算，运算时从最里面的括号开始计算 。
如 4x^4 - 2x^3 + x - 1 = (((4x - 2)x - 0)x + 1)x - 1

double polynomial_2(double A[], int n, double x)
{
    int i;
    double result = A[n - 1];
    for(i = n - 2;i>=0;i--)
    {
        result = result * x + A[i];
    }
    return result;
}

运行这两种算法：

#include <iostream>
#include <cmath>
using namespace std;

double polynomial_1(double A[], int n, double x)
{
    int i;
    double result = A[0];
    for (i = n - 1; i >= 1; i--)
    {
        result += A[i] * pow(x, i);
    }
    return result;
}

double polynomial_2(double A[], int n, double x)
{
    int i;
    double result = A[n - 1];
    for(i = n - 2;i>=0;i--)
    {
        result = result * x + A[i];
    }
    return result;
}

//4x^4-2x^3+x-1 = (((4x - 2)x - 0)x+1)x - 1
int main()
{
    double A[] = { -1,1,0,-2,4 };
    cout << polynomial_1(A, 5, 1)<<endl;
    cout << polynomial_2(A, 5, 1) << endl;
}
